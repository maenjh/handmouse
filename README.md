# HandMouse 🖐️🖱️

웹캠을 이용하여 손 제스처로 마우스를 제어하는 Python 프로그램입니다.

## 기능

- **핸드 트래킹**: MediaPipe Hands를 사용한 실시간 손 감지
- **마우스 이동**: 검지 손가락 끝 위치로 마우스 커서 제어
- **마우스 클릭**: 검지와 중지 손가락을 가까이 하면 클릭
- **스무딩**: 손 떨림 방지를 위한 부드러운 마우스 이동
- **좌표 변환**: 웹캠 화면을 모니터 전체 해상도로 매핑

## 설치 방법

### 1. 필요한 라이브러리 설치

```bash
pip install -r requirements.txt
```

### 2. 프로그램 실행

```bash
python virtual_mouse.py
```

## 사용 방법

1. 프로그램을 실행하면 웹캠이 활성화됩니다.
2. 손을 웹캠 앞에 두고 **검지 손가락**을 움직여 마우스 커서를 제어합니다.
3. **검지와 중지 손가락을 가까이** 하면 마우스 클릭이 실행됩니다.
4. **'q' 키**를 누르면 프로그램이 종료됩니다.

## 화면 표시

- **노란색 원**: 검지 손가락 끝 위치
- **보라색 원**: 중지 손가락 끝 위치
- **노란색 선**: 두 손가락 사이의 거리
- **흰색 사각형**: 사용 가능한 제어 영역
- **텍스트 정보**: 거리, 클릭 상태, 화면 좌표 등

## 설정 조정

`virtual_mouse.py`의 `main()` 함수에서 다음 파라미터를 조정할 수 있습니다:

- `frame_margin`: 프레임 가장자리 마진 (기본값: 0.1 = 10%)
- `smoothing_factor`: 스무딩 계수 (0.0 ~ 1.0, 기본값: 0.7)
- `click_threshold`: 클릭 감지 임계값 픽셀 (기본값: 40.0)

## 주의사항

- 웹캠이 정상적으로 연결되어 있어야 합니다.
- 조명이 충분한 환경에서 사용하는 것이 좋습니다.
- 손이 웹캠 화면 밖으로 나가면 마우스 제어가 중단됩니다.
